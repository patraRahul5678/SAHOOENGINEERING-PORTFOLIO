# Sahoo Engineering — Official Website

A modern, responsive business website built for **Sahoo Engineering**, a trusted civil and mechanical construction contractor based in Odisha, India.

> Developed during my internship at **Smartnex Technologies Pvt Ltd**

---

## About the Project

This website serves as the digital presence for Sahoo Engineering, showcasing their construction services, achievements, and contact information. It features smooth animations, parallax effects, and a fully responsive layout.

## Pages & Sections

- **Hero** — Full-screen parallax banner with animated geometric shapes and quick stats
- **About** — Company overview with 10+ years of experience, ISO certification, and key features
- **Services** — 6 construction services including Excavation, Hard Rock Blasting, RCC, PCC, Reinforcement Work, and Formwork & Shuttering
- **Stats** — Animated counters showcasing 150+ projects, 12m max excavation depth, 100% safety record, and 10+ years of experience
- **Contact** — Inquiry form with contact details (phone, WhatsApp, email, location)

## Tech Stack

| Technology | Purpose |
|---|---|
| React 19 | UI framework |
| Vite | Build tool & dev server |
| Tailwind CSS | Styling |
| Framer Motion | Animations & transitions |
| React Icons | Icon library |

## Getting Started

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```

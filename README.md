## 📖 About the Project

This website serves as the official digital presence for **Sahoo Engineering**, showcasing the company's expertise, achievements, and contact information.

The objective was to build a professional, visually appealing, and responsive website that reflects the organization's credibility while enabling potential clients to easily explore its services and get in touch.

The website incorporates smooth animations, parallax effects, interactive components, and a mobile-first responsive design to deliver an engaging user experience across all devices.

## 🚀 Getting Started

### Prerequisites

Make sure you have installed:

* Node.js (v18 or later recommended)
* npm

### Installation

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open your browser and navigate to:

```text
http://localhost:5173
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
